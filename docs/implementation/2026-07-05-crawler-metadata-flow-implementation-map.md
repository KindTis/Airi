## 구현 가정

- 사용자가 `master 브랜치에서 바로 작업`을 명시했으므로 `better-run-impl`의 기본 브랜치 `main` 지침보다 사용자 지시를 우선한다.
- `IOneFourOneJavCrawlerSession`은 `NavigateToAsync`, `TryGetMetadataAsync`, `TryGetThumbnailUrlAsync`를 제공하는 read/write browser session 계약으로 해석한다. provider interface는 read-only current session만 노출하고, concrete `CrawlerSessionProvider`만 `SetSession`을 가진다.
- `OneFourOneJavMetaSource`는 thumbnail 다운로드를 위해 주입 가능한 `HttpClient`를 받되, ChromeDriver/session 수명은 전혀 소유하지 않는다.
- `MainViewModel.StartCrawlerAsync`는 UI command에서 호출되는 private lifecycle method로 유지하고, public command surface는 `StartCrawlerCommand`로 유지한다.
- `OperationCanceledException` 전파 요구는 cancellation token을 받는 helper/source/factory/start path에 적용한다. `StartCrawlerCommand`와 bulk command는 계획서대로 `CancellationToken.None`을 사용한다.
- 수동 smoke test는 GUI와 실제 ChromeDriver가 필요하므로 자동화 검증과 별도로 기록한다.
